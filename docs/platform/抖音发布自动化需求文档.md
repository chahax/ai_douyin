# 抖音视频发布自动化 — 需求文档

## 1. 背景

ai_douyin 项目已完成内容生成流水线（TTS配音 + 视频拼接合成），现需要补全最后一步：将合成好的视频通过浏览器自动化发布到抖音平台。

项目根目录：`D:\IT\ai_douyin`

## 2. 目标

实现抖音创作者后台的**全自动化视频上传发布**，无需人工干预浏览器操作。

## 3. 功能需求

### 3.1 核心功能

| 功能 | 描述 |
|------|------|
| 视频上传 | 自动注入视频文件到抖音上传组件 |
| 进度等待 | 等待视频上传完成（进度条消失） |
| 标题填写 | 自动填写视频标题 |
| 描述填写 | 自动填写视频描述（可选） |
| 话题标签 | 自动添加话题标签（多个 #tag） |
| 封面上传 | 自动上传封面图片（可选） |
| 发布触发 | 点击发布按钮 |
| 结果提取 | 提取发布后的视频ID和链接 |

### 3.2 已有的基础设施

- **浏览器会话管理**：`src/platform_adapter/browser_session.py` — 已实现 Playwright 浏览器启动、登录态保存
- **发布请求模型**：`src/platform_adapter/models.py` — `PublishRequest` / `PublishResult` 数据结构已定义
- **登录态**：`data/browser/douyin/storage_state.json` — 已保存，无需每次登录
- **上传页 URL**：`https://creator.douyin.com/creator-micro/content/upload`

### 3.3 待实现模块

**主文件**：`src/platform_adapter/publish_workflow.py`

现有状态：骨架代码（仅校验+登录检测），需补全 `_do_publish()` 及各分步方法。

## 4. 输入输出

### 4.1 PublishRequest（已有）

```python
@dataclass
class PublishRequest:
    video_path: str          # 必填，视频文件路径
    title: str               # 必填，标题
    description: str = ""   # 可选，描述
    hashtags: List[str] = field(default_factory=list)  # 可选，话题标签（不含#）
    cover_path: Optional[str] = None  # 可选，封面图路径
    scheduled_at: Optional[str] = None  # 可选，定时发布
    extra_metadata: Dict[str, str] = field(default_factory=dict)
```

### 4.2 PublishResult（已有）

```python
@dataclass
class PublishResult:
    success: bool
    status: str               # published / login_required / upload_failed / error
    platform: str = "douyin"
    post_id: str = ""         # 视频ID
    publish_url: str = ""     # 发布后链接
    message: str = ""
```

### 4.3 命令行接口（已有）

```bash
python main.py douyin-publish ^
  --video "path/to/video.mp4" ^
  --title "视频标题" ^
  --desc "视频描述" ^
  --tags "励志,成长,正能量" ^
  --cover "path/to/cover.jpg"
```

## 5. 约束与边界

1. **登录态**：使用已保存的 `storage_state.json`，无需每次重新登录
2. **无 DRM 拦截**：仅上传用户自己创作的视频
3. **网络稳定**：上传超时时间 120 秒
4. **发布按钮点击后**：必须等待页面跳转或成功提示，再提取 post_id
5. **错误处理**：上传失败、页面元素找不到、发布被拒等场景均需返回有意义的 status 和 message
6. **发布频率**：不限制，由调用方控制

## 6. 验收标准

- [ ] 命令 `python main.py douyin-publish --video X --title Y` 能完整跑完流程
- [ ] 视频成功出现在抖音创作者后台"作品"列表中
- [ ] `PublishResult.post_id` 和 `publish_url` 能正确返回
- [ ] 错误场景（未登录、文件不存在、上传失败）能正确返回对应 status
