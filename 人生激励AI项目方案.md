## 一、项目概述

### 1.1 项目定位

**产品名称**: 「智悟人生」/「AI心灵导师」/「每日顿悟」

**核心价值**:
- 将经典书籍智慧转化为现代人易接受的短视频内容
- 提供7×24小时的AI人生咨询服务
- 打造"内容消费+互动陪伴"的双轮驱动模式

### 1.2 目标用户

- **核心人群**: 25-40岁职场人群
- **痛点**: 职业发展迷茫、情感困扰、焦虑内耗、缺乏行动力
- **需求**: 碎片化时间获取人生智慧、低成本情感支持、可执行的建议

### 1.3 三大核心模块

```
┌─────────────────────────────────────────────────────────┐

├─────────────────┬─────────────────┬─────────────────────┤
```

### 3.2 书籍解析与知识提取

**核心逻辑**:

```python
class BookProcessor:
    def __init__(self):
        self.llm = DeepSeekClient()  # 或 Claude/GPT-4

    def process_book(self, book_path: str) -> List[WisdomChunk]:
        """将书籍处理为智慧片段"""
        # 1. 文本提取
        raw_text = self.extract_text(book_path)

        # 2. 语义分块（保持上下文完整）
        chunks = self.semantic_split(raw_text)

        # 3. AI提炼人生智慧
        wisdom_list = []
        for chunk in chunks:
            wisdom = self.llm.extract_wisdom(
                prompt=f"""
                从以下文本中提取1-3条人生智慧，要求：
                - 适合25-40岁职场人群
                - 有具体 actionable 的建议
                - 情感共鸣强

                文本：{chunk}

                输出JSON格式：
                {{
                    "theme": "主题标签",
                    "core_wisdom": "核心道理（15字内）",
                    "elaboration": "详细阐述（50字）",
                    "actionable_tip": "具体行动建议",
                    "emotion_tag": "情感标签",
                    "suitable_scene": "适用场景"
                }}
                """
            )
            wisdom_list.append(wisdom)
        return wisdom_list
```

### 3.3 视频生成引擎

**技术方案**: 图片轮播/固定背景 + AI配音 + 动态字幕（低成本高效率）

```python
class VideoGenerator:
    def __init__(self):
        self.tts = DoubaoTTS()  # 或 Azure TTS（情感更丰富）
        self.ffmpeg = FFmpegWrapper()

    def generate_video(self, wisdom: WisdomChunk, style: VideoStyle) -> str:
        """
        生成短视频
        style: 禅意风/职场风/治愈系/热血风 等
        """
        # 1. 生成口播文案（15-45秒）
        script = self.generate_script(wisdom)

        # 2. TTS合成（带情感参数）
        audio_path = self.tts.synthesize(
            text=script,
            voice=style.voice_type,      # 温柔/坚定/激励
            speed=0.9,                    # 稍慢，显沉稳
            emotion="inspirational"
        )

        # 3. 获取背景素材
        bg_images = self.select_background(style.theme, count=3)

        # 4. 生成动态字幕（关键：卡拉OK式逐字高亮）
        subtitle_path = self.generate_karaoke_subtitle(script, audio_path)

        # 5. 视频合成
        video_path = self.ffmpeg.compose(
            images=bg_images,           # 3张图轮播，每张5秒
            audio=audio_path,
            subtitles=subtitle_path,
            bgm=style.background_music,  # 轻音乐/白噪音
            transitions="fade",         # 淡入淡出
            output_format="1080x1920"   # 9:16 竖屏
        )

        return video_path
```

## 六、核心AI Prompt设计

### 6.1 人设定义（System Prompt）

```markdown
你是"智悟"，一位温和而智慧的AI人生导师。

【核心特质】
- 像一位读过千本书的人生顾问，既有理论深度，又懂现实困境
- 说话不疾不徐，善于用故事和比喻让道理易懂
- 不居高临下，而是"我也曾经历过"的平等对话
- 每次回复控制在100-150字（移动端阅读舒适）

【知识库】
- 精通东西方哲学、心理学、管理学经典
- 擅长将《道德经》《庄子》《原则》《纳瓦尔宝典》等智慧转化为现代语境
- 熟悉认知行为疗法、正念、习惯养成等实用心理学

【禁忌】
- 不说教、不评判、不给绝对化建议
- 不替代专业心理咨询（识别危机情况需引导至专业帮助）
- 不泄露用户隐私，不在回复中暴露AI身份

【回复结构】
1. 共情确认（让用户感到被理解）
2. 智慧引用（用书籍/哲理点醒）
3. 行动建议（小而具体的下一步）
4. 开放提问（引导深入交流）
```

### 6.2 书籍智慧提取Prompt

```markdown
角色：资深编辑+人生教练
任务：从书籍内容中提取适合短视频传播的人生智慧

输入文本：{book_chunk}

处理步骤：
1. 识别核心观点（用一句话概括）
2. 找出最有冲击力的金句（适合做成标题）
3. 转化为"痛点-解药"结构：
   - 现代人常见困境（引发共鸣）
   - 书籍提供的视角（认知升级）
   - 可执行的改变（行动价值）
4. 标注情感标签（治愈/警醒/激励/释然）
5. 匹配适用场景（职场/情感/成长/健康）

输出JSON格式：
{
  "title": "短视频标题（15字内，有悬念）",
  "hook": "前3秒抓人话术",
  "core_message": "核心道理",
  "elaboration": "详细阐述（80字）",
  "actionable": "具体行动建议",
  "quote": "书中金句",
  "emotion": "情感标签",
  "scene": "适用场景",
  "related_books": ["相关推荐书籍"]
}
```

## 十、风险与应对

| 风险类型 | 具体风险 | 应对措施 | 责任人 |
|---------|---------|---------|--------|
| **合规风险** | 平台政策变化，AI内容需标识 | 严格遵守《生成式AI服务管理办法》，视频标注"AI生成" | 运营 |
| **内容风险** | 被判定为"鸡汤"或"低质" | 建立内容审核机制，确保有具体actionable建议 | 产品 |
| **用户风险** | 用户心理危机（自杀倾向） | 设置危机词库自动识别，转人工+提供求助热线 | 技术 |
| **技术风险** | API成本暴涨或限流 | 设置每日调用上限，非实时任务离线批处理 | 技术 |
| **版权风险** | 书籍内容版权争议 | 优先使用公版书，现代书籍控制引用比例<10% | 法务 |
| **竞争风险** | 同类账号模仿 | 建立书籍库壁垒，打造独特AI人设 | 运营 |

### 10.1 危机处理SOP

**识别到自杀/自残关键词时**：

1. 立即停止AI自动回复
2. 发送安抚消息："谢谢你愿意分享这些，这很重要。我这边有专业的心理咨询师可以帮你，24小时热线：400-xxx-xxxx"
3. 标记用户，通知运营人工介入
4. 记录对话日志，保留证据
